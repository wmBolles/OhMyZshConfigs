# OhMyZshConfigs
## For Ubuntu !

```
cd ~
git clone git@github.com:wmBolles/OhMyZshConfigs.git
cd OhMyZshConfigs/
chmod +x TheF.sh && chmod +x settings/*.sh && bash TheF.sh
```
